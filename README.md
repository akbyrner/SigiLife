# SigiLife
An app for creating and sharing magically imbued sigils


Naming Conventions-

If a SubComponent would naturally be titled starting with the letter 'L', the 'l' from sigil is 
dropped to create a pascal case 'SigiLxxx' component. If the Subcomponent would not naturally start with the letter L, the component is named pascal case 'SigilXxxx'.
